# noAI Search
Removes all of the AI stuff from Google, Bing, Yahoo Search, and DuckDuckGo!
![logo](https://raw.githubusercontent.com/TecoTurtle/noAI-search/refs/heads/main/icon.svg)

**Supported Search Engines**  
-
- Google  
- Bing  
- Yahoo Search  
- DuckDuckGo  

**Features**  
-
- Removes AI Overview from Google  
- Removes all AI Mode buttons from Google  
- Removes Copilot features from Bing  
- Removes AI Chat button on Yahoo  
- Removes Search Assist from DuckDuckGo  
- Removes Duck.ai and Search Assist buttons from DuckDuckGo
